===================================
Feature: #36 - Add price categories
===================================

See :issue:`36`

Description
===========

This feature allow different price categories for an event date. Each event date can have several price categories
with special prices as well. In the event date a checkbox allows the seat handling in the event date or for each
price category itself.
In frontend the customer see a selector for the price category. Some new JavaScript shows the price for the selected
price category.
