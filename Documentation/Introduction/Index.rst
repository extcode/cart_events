Introduction
------------

Cart is a small but powerful extension which "solely" adds a shopping cart to your TYPO3 installation.
The extension allows you to create coupons, products with or without variants, special prices.
CartEvents provides an own data storage for events. Events can be offered via a list and detail view and can be purchased via cart function of the Cart extension.
